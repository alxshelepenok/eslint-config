# @alxshelepenok/eslint-config 

[![circleci][1]][2]
[![downloads][3]][4]

Shared eslint config for my projects.

# Install

```
yarn add @alxshelepenok/eslint-config --dev
```

# Example

```json
{
  "extends": "@alxshelepenok/eslint-config"
}
```
